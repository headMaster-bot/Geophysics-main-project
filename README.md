# GeoPhysics Frontend
reinstall node_modules by using npm i
cd into the project folder using cd Geophysics-main-project
start the project with the command npm start

This is the frontend application for the GeoPhysics Project Management System built with React and Redux Toolkit.

## Features

- Project management
- Epic and story tracking
- Survey management
- Draft saving
- Responsive dashboard UI
- State management with Redux Toolkit

## Tech Stack

- React.js
- Redux Toolkit
- React Router
- Axios
- Tailwind CSS
- SweetAlert2
